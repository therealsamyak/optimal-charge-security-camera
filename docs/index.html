Optimal Charge Security Camera

Intelligent security camera with energy-aware model selection

Quick Start

Get running in 30 seconds

./scripts/start.sh

Default webcam processing with intelligent controller

./scripts/start.sh --controller hybrid --verbose

Hybrid controller with detailed logging

./scripts/tests.sh

Run comprehensive test suite

Requirements: Python 3.8+, UV package manager, webcam access

Goals

  • Create a battery-powered security camera that runs local image recognition
  • Optimize energy usage based on clean energy availability
  • Balance model accuracy with power consumption
  • Make intelligent decisions about charging vs processing

Specific Aims

  • Implement context-aware battery management system
  • Integrate clean energy API for real-time grid information
  • Develop controller that switches between YOLOv10 models based on conditions
  • Create simulation environment using Jetson hardware
  • Build software battery simulation for testing

Approach

We're building a system that considers multiple factors to optimize security camera performance:

  • Hardware: Jetson device simulating security camera
  • Models: YOLOv10n (lightweight) and YOLOv10s (more accurate)
  • Controller: Decision engine that evaluates:
    • Current battery percentage
    • Clean energy availability from external API
    • User-defined metrics (accuracy, latency, frequency)
  • Battery Simulation: Software model where tasks consume energy and charging can be toggled

Deliverables

  • Functional controller system for model selection
  • Battery simulation software
  • Clean energy API integration
  • Jetson-based camera simulation
  • Performance metrics and optimization results
  • Documentation and code repository

Timeline

Each step: ~1 week

  • Set up Jetson environment and basic camera simulation
  • Implement battery simulation and YOLOv10 models
  • Integrate clean energy API and develop controller logic
  • Test and optimize decision-making algorithms
  • Final testing, documentation, and performance analysis
